# Layout Patterns

Expressive design leverages these popular layout trends to create structured, engaging, and functional user interfaces.

## 1. Bento Grid
A modular, box-based grid (often with rounded corners) that displays various pieces of content—like widgets, cards, or features—in a structured, "lunchbox" style. Very popular for feature showcases and dashboards.

## 2. Masonry Layout
A grid layout where elements (cards or images) are stacked vertically based on available vertical space, resulting in an uneven bottom edge. Ideal for portfolios and content feeds (e.g., Pinterest).

## 3. Card-Based Layout
An interface design that organizes information into distinct, rectangular containers called "cards." Each card typically acts as a single, cohesive unit of information or an entry point for a specific action.

## 4. Split-Screen Layout
A layout that divides the screen into two distinct vertical or horizontal sections, often contrasting two types of content, such as a large hero image on one side and text/form fields on the other.

## 5. Dashboard Sidebar Layout
A standard administrative design that features a persistent navigation column on the left (or right) side, providing quick access to primary app sections while maintaining a dedicated "workspace" area in the center.

## 6. Hero Section Layout
A high-impact, full-width top section of a website or app meant to capture user attention immediately. It usually contains a bold headline, a call-to-action (CTA), and a large visual element.

## 7. Multi-Column Grid
A traditional design system approach using vertical columns (often 12-column grids) to align content, ensuring consistency and balance across different screen sizes.

## 8. Single-Page (Landing) Layout
A design style where all primary content is organized onto one long, scrollable page, often using anchor links for navigation rather than traditional multi-page loads.

## 9. Tabbed Layout
A pattern that organizes content into multiple interactive tabs. Ideal for compact spaces where you need to switch between categories or views without leaving the current screen.

## 10. Sticky Header Layout
A structural pattern where the site's navigation header stays fixed to the top of the viewport as the user scrolls, ensuring that navigation is always accessible without returning to the top of the page.

## 11. Master-Detail Layout
A classic pattern (common in email clients and settings menus) where a navigation list occupies a narrow column on one side, and the content for the selected item appears in a larger, adjacent main panel.

## 12. Z-Pattern Layout
A design that directs the user’s eye across the top, diagonally down to the bottom corner, and then across the bottom. This is widely used for marketing landing pages to ensure key call-to-action (CTA) buttons are seen.

## 13. Full-Bleed Layout
A high-impact visual design where images, videos, or background colors extend to the very edges of the screen, removing all visible margins and borders for an immersive experience.

## 14. Asymmetrical Layout
A departure from rigid grids, this layout uses off-balance elements, varying container sizes, and creative negative space to provide a more artistic or modern "editorial" look.

## 15. Accordion Layout
A vertical stacking of headers that, when clicked, expand to reveal additional information. This is used extensively to keep long pages clean and reduce cognitive load.

## 16. Timeline/Vertical Stepper Layout
Used for progress tracking, transaction histories, or onboarding flows. It connects distinct items or steps with a vertical line, creating a clear sense of sequence and progression.

## 17. Magazine (Editorial) Layout
A style inspired by print media that focuses on hierarchy through large, bold typography, varied column widths, and overlapping elements rather than a standard uniform grid.

## 18. Centric (Focus) Layout
A minimalist design that places the primary element (often a product image or a login form) in the dead center of the screen, removing background distractions to maximize user focus.

## 19. F-Pattern Layout
Based on eye-tracking research, this layout places the most important content along the top and down the left side, mimicking the way users naturally scan text-heavy pages.

## 20. Modal/Overlay Layout
A layout pattern that temporarily suspends the main interface by layering a focused panel on top of the content. This is commonly used for logins, quick views, or confirmation requests without forcing a full page navigation.

## 21. Command Center (Data-Dense) Layout
A specialized dashboard layout designed for high-performance users (e.g., financial traders, DevOps engineers). It prioritizes screen real estate for maximum information density, often using micro-charts and tiny text labels to avoid page refreshes.

## 22. Studio/IDE Layout
A complex, multi-pane layout modeled after professional software like Figma or Adobe Creative Cloud. It features a central working "canvas" surrounded by "tool pallets," property inspectors, and layer panels, often allowing for adjustable/resizable zones.

## 23. Bottom-Sheet (Reachability) Layout
A mobile-specialized layout that anchors primary interactions (or auxiliary information) to the bottom of the screen. This is designed for one-handed phone use, allowing users to interact with content without reaching to the top of the device.

## 24. Scrollytelling Layout
A narrative-driven layout where the design reacts to the user's scroll. As the user moves down the page, elements animate, fade, or move to tell a story or explain a complex feature. This is frequently used for high-end marketing sites and product launches.

## 25. Mega-Menu Layout
A specialized navigation structure for large e-commerce or enterprise sites. Instead of a simple dropdown, the menu expands into a large, multi-column panel that categorizes links with icons, featured images, and promotional banners.

## 26. Comparison (Diff) Layout
A specialized side-by-side or "before-and-after" view. It is used in software for code diffing, price comparison tools, or product feature alignment, typically allowing users to toggle highlighting for differences.

## 27. Infinite Canvas Layout
Unlike standard web pages with rigid borders, this layout simulates a limitless plane. Users can drag, zoom, and pan across the space (common in whiteboarding or collaborative design tools like Miro or FigJam).

## 28. Stacked-Card (Story) Layout
A mobile-first design that treats content like a deck of cards. Users swipe horizontally or vertically to progress through content. It is the core layout pattern for modern apps like TikTok, Instagram Stories, or Tinder.

## 29. Floating Action (FAB) Layout
A specialized mobile layout built around a "Floating Action Button." The interface is designed to push all primary actions into a single, prominent, thumb-accessible button that, when pressed, expands into a menu or sub-actions.

## 30. Split-Pane (Master-Detail) Email/Chat Layout
A highly functional layout for communication tools. It maintains a persistent list of contacts or folders on the left, while the main viewing area stays fixed to the right, allowing the user to switch conversations instantly without reloading the entire page.
